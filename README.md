# Этот бот может обрабатывать результаты экспериментов и вполне годиться для первого курса естественнонаучных факультетов
## Математика
### Аппроксимация.
Бот умеет считать аппроксимирующую функцию, при чём достаточно разнообразную. Достаточно прислать её боту в сообщении и он построит аппроксимант с погрешностью. Также для этого кончено нужны экспериментальные данные которые в боте также удобно хранятся по папкам.
### Случайные погрешности.
Бот может считать случайные погрешности измерений. Например, вы измерили длину чего-либо несколько раз. Бот отправит среднее значение и +/- диапазон возможных значений.
## Хранение данных.
Бот хранит данные в папках, которые вы можете сами создать. В каждой папки есть измерения и серии измерений. Каждое измерение можно объединить с другими в серию. Это нужно для того, чтобы была возможность строить аппроксимацию с точками, у которых есть случайная погрешность. После объединения измерений в серию они представляют собой набор координат с +/- диапазоном значений. Папки, записи, серии можно удалить. Лимит папок - 5.
## Почему может быть полезно?
Популярнее всего у школьников и студентов среди мессенджеров Телеграм. Помимо отправки сообщений, в нём есть возможность вести блог и пользоваться ботами. Бот в Телеграме можно открыть прямо на телефоне, а взаимодействие с ним происходит с помощью сообщений и кнопок. Удобно и быстро. Так студент или школьник смоет быстро обрабатывать свои экспериментальные данные с лабораторных работ. Написал данный проект сам будучи студентом физического факультета для того, чтобы удобно использовать и не переписывать для каждой лабы разный код.

# This bot can process experimental results and is perfectly suitable for first-year students of natural science faculties
## Mathematics
### Approximation
The bot can calculate an approximating function — and quite a diverse one. You just need to send the function to the bot, and it will build an approximation with an error estimate. Of course, you also need experimental data, which the bot conveniently stores in folders.
### Random errors
The bot can calculate random measurement errors. For example, if you measured the length of something several times, the bot will provide the mean value and the +- range of possible values.
## Data storage
The bot stores data in folders, which you can create yourself.
Each folder contains measurements and measurement series.
You can combine multiple measurements into a series - this allows you to build an approximation using points with random errors.
Once combined, the series represent a set of coordinates with +- value ranges.
Folders, records, and series can be deleted. The limit is 5 folders per user.
## Why is it useful?
Telegram is one of the most popular messengers among students and school pupils.
In addition to messaging, it allows blogging and using bots.
A Telegram bot can be opened directly on your phone, and you interact with it using buttons and messages — fast and convenient.
This makes it easy for students to process experimental data from laboratory works.
I developed this project as a physics student myself, to make it convenient to use and to avoid rewriting separate code for each lab work.
